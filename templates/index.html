Quote Generator

Use your own quotes to create a custom quote API. Create a collection, then add quotes to that collection. You can then use the random endpoint to power your own quote machine.

Built by Daniel Jones for freeCodeCamp students.

API

Get all quotes in a collection.

{{ config.APP_URL }}/api/collection/<collection_name>

Get a random quote from a collection.

{{ config.APP_URL }}/api/collection/<collection_name>/random

Get a single quote by it's ID.

{{ config.APP_URL }}/api/quote/<quote_id>
